⚙ Configurar conexión a Drive
Para guardar manifiestos firmados en Google Drive necesitas desplegar el script en Google Apps Script y pegar la URL aquí.
Pasos:
1. Abre script.google.com → Nuevo proyecto
2. Pega el contenido del archivo google-apps-script.js
3. Edita FOLDER_ID y SHEET_ID con tus IDs
4. Implementar → Nueva implementación → Aplicación web
5. Copia la URL y pégala abajo
